# Ovice Reaction Maker

Text to Animated GIF generator for Ovice and other virtual offices.

## Features
- **Live Preview**: See your reaction frame by frame.
- **Customizable**: Change text, colors, background (transparent supported), and animation type.
- **Client-Side Generation**: Privacy focused, no server uploads.
- **Optimized for Reactions**: Creates small, looped GIFs perfect for chat or virtual avatars.

## Tech Stack
- React
- Vite
- Canvas API
- gifenc (for GIF encoding)

## Development
```bash
npm install
npm run dev
```
