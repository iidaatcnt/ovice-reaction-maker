# Ovice Reaction Maker 🚀

Ovice 等のバーチャルオフィスやコミュニケーションツールで使える「動くリアクション」を簡単に作成できるツールです。

## 使い方 📖

1. **テキストを入力**: 表示したいメッセージを入力します（例: YES）。
2. **デザインを選択**:
   - **アニメーション**: 動きの種類を選びます。
   - **背景パターン**: 背景の模様を選びます。
   - **キャンバスサイズ**: 使用場所に合わせてサイズを選びます。
3. **色を調整**: テキストの色や、背景（透過または色付き）を設定します。
4. **ダウンロード**: 「GIFを生成してダウンロード」ボタンを押すと、GIFファイルが保存されます。

## 機能の特徴 ✨

- **透明背景対応**: Ovice のマップ上で自然に見える透明 GIF が作成できます。
- **文字サイズ調整**: 文字数に合わせて `+` `-` ボタンで微調整が可能です。
- **スマホ対応**: レスポンシブデザインでどこからでも作成できます。

## 開発者向け 🛠️

このプロジェクトは Next.js で構築されています。

### 起動方法
```bash
npm install
npm run dev
```
